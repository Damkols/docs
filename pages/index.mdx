# Sound Docs

### Sound Protocol

The Sound Protocol is a permissionless, open-source, modular smart contract framework for musicians and creators.

[Sound Protocol docs](/protocol/overview/intro)

### Sound SDK

The Sound SDK is a Typescript library for building applications on the Sound Protocol.

[Sound SDK docs](/sdk).

## Sound API

Our official GraphQL API serves indexed event data from Sound Protocol, including metadata like image and audio URLs.

It is a convenient way to integrate Sound data directly into your own tech stack.

[Sound API docs](/sound-api).
