# Sound Protocol

Sound Protocol is a generalized platform for flexible and efficient creation of NFT drops. It consists of a factory, `SoundCreatorV1`, which deploys minimal proxies of a 721a NFT contract, `SoundEditionV1`, which can optionally implement a set of module contracts responsible for functionality like minting and NFT metadata.

## Core contracts

These contracts form the core of the system.

- [SoundCreatorV1](/protocol/core/sound-creator-v1) - A factory that deploys Sound editions.
- [SoundEditionV1](/protocol/core/sound-edition-v1) - The 721a NFT contract.
- [IMinterModule](/protocol/core/i-minter-module) - The interface for minter module contracts.
- [IMetadataModule](/protocol/core/i-metadata-module) - The interface for metadata module contracts.

## Module contracts

These are the current set of modules that can be optionally implemented by Sound Editions.

### Metadata modules

- [GoldenEggMetadata](/protocol/modules/golden-egg-metadata) - Implements the Golden Egg used by sound.xyz.

### Minter modules

- [EditionMaxMinter](/protocol/modules/edition-max-minter) - A minimalist fixed-price public minting module. Can implement single-schedule range mints.
- [RangeEditionMinter](/protocol/modules/range-edition-minter) - Implements any number of fixed-price range mint schedules for an edition.
- [FixedPriceSignatureMinter](/protocol/modules/fixed-price-signature-minter) - Implements fixed-price signature-based minting.
- [MerkleDropMinter](/protocol/modules/merkle-drop-minter) - Implements fixed-price Merkle proof minting (allow lists).
